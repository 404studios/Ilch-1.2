Informationen zur Tabelle prefix_modules
In dieser Tabelle sind alle Module eingetragen, für die es ein Adminbereich gibt oder für die Rechte verteilt werden müssen.

name
Name, der im Adminbereich in der Navigation erscheint und bei der Rechtvergabe

url
Ist die URL zum Modul über den Adminbereich

gshow
Definiert, ob User das Modul im Adminbereich sehen, wenn diese die Rechte für das Modul besitzen

ashow
Definiert, ob das Modul nachträglich installiert worden ist

fright
Stellt fest, ob Rechte für das Modul vergeben werden können
